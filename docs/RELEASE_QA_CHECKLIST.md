# Release QA Checklist v1.0\n\n## 릴리즈 전 최종 검증 (30개 항목)\n\n### 📱 기본 기능 (5개)\n\n- [ ] **1. 앱 시작**: 앱을 실행하면 정상적으로 로드되고 튜토리얼이 표시됨\n- [ ] **2. 튜토리얼**: 4단계 튜토리얼을 완료하면 Home 화면으로 이동\n- [ ] **3. Privacy Policy**: 설정에서 Privacy Policy 접근 가능\n- [ ] **27. Community Guidelines**: 설정에서 Community Guidelines 접근 가능\n- [ ] **28. 데이터 초기화**: 설정에서 \"데이터 초기화\" 버튼이 작동\n- [ ] **29.
